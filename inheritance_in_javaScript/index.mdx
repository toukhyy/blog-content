Inheritance is a fundamental concept in object-oriented programming that allows objects to inherit properties and methods from parent objects. In JavaScript, inheritance can be achieved through prototype chaining.

```javascript
// Parent class
function Animal(name) {
  this.name = name;
}

// Method shared by all instances of Animal
Animal.prototype.sleep = function () {
  console.log(this.name + ' is sleeping.');
};

// Child class inheriting from Animal
function Cat(name) {
  Animal.call(this, name);
}

// Set the prototype of Cat to an instance of Animal
Cat.prototype = Object.create(Animal.prototype);

// Method specific to Cat instances
Cat.prototype.meow = function () {
  console.log(this.name + ' says meow!');
};

// Create instances of Animal and Cat
var animal = new Animal('Generic Animal');
var cat = new Cat('Tom');

// Usage
animal.sleep(); // Output: Generic Animal is sleeping.
cat.sleep(); // Output: Tom is sleeping.
cat.meow(); // Output: Tom says meow!
```

In the code sample above, we have an `Animal` class with a `sleep` method. The `Cat` class is created using the `Animal.call(this, name)` syntax, which calls the `Animal` constructor with the `this` context set to the `Cat` instance. Then, we use `Object.create(Animal.prototype)` to set the prototype of `Cat` to an instance of `Animal`. This establishes the prototype chain, allowing the `Cat` instances to inherit the `sleep` method from `Animal`.

By leveraging prototype chaining, JavaScript enables us to create a hierarchy of objects and share functionality between them through inheritance.

## 2. Higher-Order Functions in JavaScript

Higher-order functions are functions that can take other functions as arguments or return functions as their results. They are a powerful feature in JavaScript, allowing for more flexible and expressive code.

```javascript
// Higher-order function example: map
function map(array, transform) {
  var result = [];
  for (var i = 0; i < array.length; i++) {
    result.push(transform(array[i]));
  }
  return result;
}

// Function to be passed as an argument to map
function double(number) {
  return number * 2;
}

// Usage
var numbers = [1, 2, 3, 4, 5];
var doubledNumbers = map(numbers, double);
console.log(doubledNumbers); // Output: [2, 4, 6, 8, 10]
```

In the code sample above, we have a higher-order function called `map` that takes an array and a `transform` function. It iterates over each element in the array and applies the `transform` function to it, building a new array with the transformed values.

We define the `double` function, which simply doubles a given number. We then pass `double` as an argument to `map` along with an array of numbers. The `map` function applies the `double` function to each element in the array, resulting in a new array where each number is doubled.

Higher-order functions like `map` provide a concise and reusable way to perform common operations on arrays, such as filtering, transforming, or reducing their elements. They promote code modularity and can significantly enhance the readability and maintainability of your JavaScript code.
